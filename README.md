# for_fun

This repository contains a small static site. To publish it to GitHub Pages:

1. Create a GitHub repository and push this folder as the `main` branch.
2. Actions will run and deploy the site to GitHub Pages automatically.

Quick push commands:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/<your-username>/<repo>.git
git push -u origin main
```

After the push, open the repository's Pages settings to confirm the site URL.
